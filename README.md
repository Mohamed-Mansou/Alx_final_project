# Todo-List
for alx project
### Features
* Add new task
* Mark task as completed / uncompleted
* Edit task
* Delete single task
* Delete completed tasks
* Delete all tasks
* ### Technologies used:
* HTML
* CSS
* JavaScript/Es6
